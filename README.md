# 🛒 Egypt E-Commerce Simulation Database

Welcome to the **Egypt E-Commerce Simulation Dataset** repository. This project provides a comprehensive, simulated relational database for an e-commerce platform operating in the Egyptian market. It is designed to be a practical playground for data professionals to practice SQL, design Business Intelligence dashboards, and build predictive models.

## 📌 Project Overview
The data mimics real-world business operations, capturing the end-to-end customer journey—from product browsing and ordering to payment processing and delivery across various Egyptian cities. 

This dataset is perfect for those looking to build strong portfolio projects in **Data Analysis**, **SQL**, and **Power BI**.

## 📂 Repository Structure
* `/data/`: Contains the 7 CSV files (`customers.csv`, `orders.csv`, `employees.csv`, etc.) representing the simulated data.
* `/sql/`: Contains SQL scripts for creating the database schema and sample analytical queries.
* `/images/`: Contains diagrams and dashboard screenshots.

## 💡 How to Use This Data
You can use this database to:
1. **Practice Complex SQL:** Write queries using `JOIN`s, Window Functions, and CTEs to extract insights like customer retention and top-performing sales reps.
2. **Build BI Dashboards:** Connect the CSV files or your SQL database to Power BI/Tableau to visualize revenue growth, shipping fees, and inventory levels.
3. **Machine Learning:** Predict order cancellations or analyze market basket trends.
